Authentication

Secure API access using AWS Cognito JWT tokens

🔑 Microsoft SSO Authentication (Recommended)
Users authenticate via Microsoft Single Sign-On (SSO). Click "Sign in with Microsoft" and authenticate with your Microsoft account (personal, work, or school). JWT tokens are automatically issued by AWS Cognito after successful Microsoft authentication.
SSO Sign In with Microsoft

Initiate Microsoft OAuth sign-in flow. Users are redirected to Microsoft login and back to the app with tokens.

JavaScript - Microsoft SSO with Amplify 
import { signInWithRedirect } from 'aws-amplify/auth';

async function signInWithMicrosoft() {
  await signInWithRedirect({
    provider: {
      custom: 'Microsoft'
    }
  });
  // User is redirected to Microsoft login page
  // After auth, user returns to /auth/callback with tokens
}

Flow: User clicks button → Redirect to Microsoft → User signs in → Redirect back to app → JWT tokens issued automatically

TOKEN Retrieve Token for API Calls

After sign-in, retrieve the JWT token to use in API requests.

JavaScript - Get Auth Token 
import { fetchAuthSession } from 'aws-amplify/auth';

async function getAuthToken() {
  const session = await fetchAuthSession();
  const token = session.tokens?.idToken?.toString();

  if (!token) {
    throw new Error('No valid session');
  }

  return token;
}

// Use token in API calls
async function fetchShipments() {
  const token = await getAuthToken();

  const response = await fetch(
    'https://h8fcclh73j.execute-api.us-east-1.amazonaws.com/prod/shipments',
    {
      headers: {
        'Authorization': `Bearer ${token}`,
        'Content-Type': 'application/json',
      },
    }
  );

  return response.json();
}
